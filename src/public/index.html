User

POST /user/signup

Este endpoint serve para realizar o cadastro do usuário no sistema. É uma requisição do tipo POST, passando as informações do cliente via body, e em caso de sucesso, retorna um JSON com o nome do utilizado no cadastro e o token de acesso.

Requisição 
                        
                        {
                            name: "joao",
                            password: 123456
                        }
Resposta caso sucesso 
                        
                        {
                            "success": true,
                            "data": {
                                "name": "joao",
                                "token": "jwt que expira após 2 horas."
                            }
                        }
Resposta caso fracasso 
                        
                        {
                            "success": false,
                            "message": "mensagem do erro que ocorreu"
                        }

Auth

GET /auth/signin

Este endpoint serve para realizar a autenticação do usuário no sistema, após a autenticação ser realizada o usuário, a partir do token, será possível realizar as operações de CRUD(Create, Read, Update e Delete) da sua lista de tarefas. É uma requisição do tipo GET, passando via body o nome de usuário e senha, e em caso de sucesso, retorna um JSON com o token de acesso.

Requisição 
                        
                        {
                            name: "joao",
                            password: 123456
                        }
Resposta caso sucesso 
                        
                        {
                            "success": true,
                            "data": {
                                "token": "jwt que expira após 2 horas."
                            }
                        }
Resposta caso fracasso 
                        
                        {
                            "success": false,
                            "message": "mensagem do erro que ocorreu"
                        }

Todo

POST /todos/

Este endpoint serve para realizar a criação de uma tarefa na lista do usuário. É uma requisição do tipo POST que recebe via body, o título, a descrição e o token válido do usuário. Caso sucesso, é retornado o id da tarefa criada.

Requisição 
                        
                        {
                            title: "Lorem ipsum dolor sit.",
                            description: "Lorem, ipsum dolor sit amet consectetur adipisicing elit. Nisi, impedit?",
                            token: "jwt do usuário autenticado"
                        }
Resposta caso sucesso 
                        
                        {
                            "success": true,
                            "data": "62def2fbdce74f4100283f29"
                        }
Resposta caso fracasso 
                        
                        {
                            "success": false,
                            "message": "mensagem do erro que ocorreu"
                        }
